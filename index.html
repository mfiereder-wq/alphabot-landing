Setup in 5 Minuten

Keine Programmierkenntnisse noetig - folge einfach diesen Schritten

1

Python installieren

Stelle sicher, dass Python 3 auf deinem Computer installiert ist.

python --version
  • Falls nicht installiert: python.org herunterladen
  • Bei der Installation "Add to PATH" aktivieren
2

Abhaengigkeiten installieren

Oeffne Terminal/Command Prompt und fuehre aus:

pip install python-binance python-telegram-bot pandas numpy
  • Dies installiert alle benoetigten Bibliotheken
  • Kann ein paar Minuten dauern
3

Datei entpacken

Extrahiere die heruntergeladene ZIP-Datei in einen Ordner.

  • Du findest: multi_strategy_bot.py
  • Oeffne die Datei mit einem Text-Editor (z.B. Notepad++)
4

API Keys eintragen

Fuelle die Config-Sektion am Anfang der Datei aus:

  • BINANCE_API_KEY - von deinem Binance Account
  • BINANCE_SECRET_KEY - ebenfalls von Binance
  • TELEGRAM_TOKEN - von @BotFather
  • TELEGRAM_CHAT_ID - deine Chat ID
5

Bot starten

Speichere die Datei und fuehre sie aus:

python multi_strategy_bot.py
  • Waehle Symbol (z.B. BTCUSDT) und Timeframe (z.B. 15m)
  • Lege dein Leverage fest (empfohlen: 5-10x)
  • Fertig! Der Bot tradet nun automatisch
